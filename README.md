# Smart Attendance Management System

A web-based attendance management system developed using HTML, CSS, JavaScript, NodeJS, and ExpressJS.

## Features

- Add and manage student records
- Mark Present/Absent attendance
- Student search functionality
- Attendance analytics dashboard
- Responsive user interface

## Tech Stack

- HTML
- CSS
- JavaScript
- NodeJS
- ExpressJS

## How to Run

```bash
npm install
npm start
```

Open:

```txt
http://localhost:3000
```

## Project Overview

This project was developed to understand frontend-backend integration, REST APIs, and basic full-stack web development concepts. The system allows users to manage student attendance records through a simple and responsive web interface.
